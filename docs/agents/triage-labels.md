# Triage labels

Use the default triage label vocabulary.

| Role | Label |
| --- | --- |
| Needs maintainer evaluation | `needs-triage` |
| Waiting on reporter | `needs-info` |
| Fully specified and ready for an AFK agent | `ready-for-agent` |
| Requires human implementation | `ready-for-human` |
| Will not be actioned | `wontfix` |

## Consumer rules

- Apply exactly these label names unless this file is updated.
- Avoid creating duplicate labels with alternate spellings.
- Preserve any product/team labels already present on an issue.
